# Portfolio Website

A modern and responsive personal portfolio website showcasing skills, projects, experience, and contact information.

## Features

- Responsive design
- Dark/Light mode
- Smooth scrolling animations
- Skills section
- Projects section
- Experience section
- Certificates section
- Contact form with email support
- SCSS-based styling

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- SCSS
- Node.js
- Express.js

## Folder Structure


css/
js/
scss/
server/
index.html
package.json


## Installation

bash
npm install


## Run Server

bash
node server/server.js
